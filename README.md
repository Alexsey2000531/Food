# Food

Веб-приложение для расчёта и заказа питания.

## Описание

Это одностраничное приложение позволяет:
- Просматривать меню
- Рассчитывать калории
- Оформлять заказ

Для корректной работы требуется запускать:
- **json-server** (имитация backend API)
- **Live Server** (запуск фронтенда)

---

## Быстрый старт

### 1. Клонируйте репозиторий

```sh
git clone https://github.com/Alexsey2000531/Food.git
cd Food
```

### 2. Установите json-server (если не установлен)

```sh
npm install -g json-server
```

### 3. Создайте файл `db.json` в корне проекта

Пример содержимого:
```json
{
  "menu": [
    {
      "id": 1,
      "title": "Фитнес",
      "description": "Меню для активных людей",
      "price": 500
    }
  ],
  "orders": []
}
```

### 4. Запустите json-server

```sh
json-server --watch db.json --port 3001
```
API будет доступно по адресу: http://localhost:3001

### 5. Запустите Live Server

- Откройте проект в редакторе (например, VS Code)
- Кликните правой кнопкой по `index.html` → **Open with Live Server**
- Откроется страница, обычно по адресу http://127.0.0.1:5500

---

## Использование

- Откройте приложение в браузере (адрес выдаст Live Server)
- Убедитесь, что json-server работает на http://localhost:3000
- Приложение будет обращаться к API для получения меню и отправки заказов
